# SARPreProcessing_dev

This is a simple Python script that uses ESA Snap gpt.exe to run a xml file with the pre processing steps for SAR images.

This script was intended to run via terminal, for that, you need to have ESA Snap software installed and use the graph builder tool to make your own pre processing workflow and use the xml file generated from the graph builder tool.

Then Python will run the gpt.exe using the xml file and process every SAR image inside the folder you specified.
